Team Creator Workflows

All three input methods update the same player team. Importing, editing manually, or rendering raw text will stay synchronized.

Pokepaste Import

Best when you already have a Pokepaste URL or ID. Import the team quickly, then fine-tune it in the other tabs if needed.

Manual Builder

Best for building one slot at a time. Use this for flexible type slots, custom notes, or small edits without touching raw text.

Raw Import

Best when you want direct Showdown-style control. Render raw text into the shared team, clear the current player, or load full `# P1` to `# P4` exports at once.
